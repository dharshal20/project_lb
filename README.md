Data Analyst Job Market Analysis
📌 Project Overview

This project explores the data job market, with a strong focus on Data Analyst roles.
The goal is to identify:

💰 High-paying skills
📈 In-demand skills
🎯 Optimal skills (high demand + high salary)

Using real-world job data, this analysis helps aspiring data analysts make data-driven career decisions.

📂 Dataset

The dataset is sourced from Luke Barousse’s Python Course, containing:

Job titles
Salaries
Locations
Required skills
❓ Key Questions Answered
What are the most in-demand skills for the top 3 data roles?
How are Data Analyst skills trending over time?
How well do Data Analyst jobs and skills pay?
What are the most optimal skills to learn (high demand + high pay)?
🛠️ Tools & Technologies Used
Python → Core analysis
Pandas → Data cleaning & manipulation
Matplotlib & Seaborn → Data visualization
Jupyter Notebook → Analysis workflow
VS Code → Development environment
Git & GitHub → Version control
🧹 Data Cleaning & Preparation
Converted date columns into proper datetime format
Transformed job_skills from string → list using ast.literal_eval
Filtered dataset to focus on United States jobs
📊 Analysis & Insights
1️⃣ Most In-Demand Skills (Top Roles)
SQL → Most demanded skill across Data Analysts & Data Scientists
Python → Highly demanded across all roles, especially Data Engineers
Cloud & Big Data (AWS, Azure, Spark) → Key for Data Engineers

👉 Insight:

Analysts need core tools (SQL, Excel, Tableau)
Engineers need specialized tech skills
2️⃣ Skill Trends (2023)
SQL → Consistently dominant
Excel → Strong growth toward end of year
Python & Tableau → Stable demand
Power BI → Gradual increase

👉 Insight:
Skill demand is dynamic, staying updated is crucial.

3️⃣ Salary Analysis
Senior Data Scientist → Highest salary potential
Senior roles → Higher pay + higher variability
Data Analysts → More stable salaries

👉 Insight:
Salary increases with:

Experience
Specialization
4️⃣ Highest Paying vs Most In-Demand Skills
💰 Highest Paying:
Specialized tools (e.g., Oracle, GitLab, Bitbucket)
🔥 Most In-Demand:
SQL
Excel
PowerPoint

👉 Insight:

High demand ≠ High salary
Balance is key
5️⃣ Optimal Skills (High Demand + High Pay)

Top skills:

Python
SQL
Tableau
SQL Server

👉 Insight:
These skills provide the best career ROI

📈 Key Learnings
Importance of data cleaning before analysis
Strong hands-on experience with Pandas & visualization tools
Understanding market-driven skill selection
⚠️ Challenges Faced
Handling missing/inconsistent data
Creating clear visualizations
Balancing detailed vs broad analysis
🎯 Conclusion

This project highlights how:

Skill demand and salary are closely related
The job market is constantly evolving
Strategic skill-building is essential

👉 Final Takeaway:
To succeed as a Data Analyst, focus on:

Core skills (SQL, Excel)
High-value skills (Python, Tableau)
